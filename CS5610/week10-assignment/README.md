# Week 10 – Vue & Angular Pages – Student

This repo contains two small single-page web apps demonstrating basic Vue and Angular usage.  

## Files

- `vue-dessert.html` – Dessert rating page implemented with Vue 2 (CDN).
- `angular-dessert.html` – Dessert rating page implemented with AngularJS 1.x (CDN).
- `style.css` – Shared styling for both pages.
